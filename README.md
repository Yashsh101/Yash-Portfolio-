## Deploy in 60 Seconds

**Vercel (recommended)**
```bash
npm i -g vercel
vercel --prod
```

**GitHub Pages**
```
Settings → Pages → Branch: main → / (root) → Save
```

**Local Preview**
```bash
open index.html        # macOS
start index.html       # Windows
```
